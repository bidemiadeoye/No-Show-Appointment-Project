# 🏥 Healthcare Appointment No-Show Analysis Program

## 📌 Overview
This project is a Python-based application designed to analyze healthcare appointment data and identify patterns in missed appointments (no-shows). It provides simple, actionable insights that can help healthcare providers improve scheduling efficiency and reduce missed visits.

---

## 🎯 Objective
The objective of this project is to:
- Analyze appointment data for no-show trends
- Provide insights based on:
  - Day of the week
  - Time of day
  - Patient age group
- Demonstrate practical use of Python in healthcare analytics

---

## ⚙️ Features
- Menu-driven command-line interface
- User-provided CSV file input
- Calculates:
  - Overall
